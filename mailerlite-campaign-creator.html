About
How this tool works and what's running behind the scenes.
What this tool does
This tool automates the weekly email campaign workflow for The RE Source — from a new YouTube upload to a fully built MailerLite draft — in about 3 minutes.
1
Fetch
Pulls the latest YouTube video title, thumbnail, description, and URL. Also fetches the matching WordPress post URL from the videos custom post type.
2
Process thumbnail
Applies the play button overlay and cuts the bottom corners into the chevron shape — entirely in the browser using the Canvas API. The processed PNG is uploaded to WordPress media automatically.
3
Review & edit
Edit the subject line, preheader, body copy, and social links. Instagram, Facebook, LinkedIn, and Podcast are pre-filled with The RE Source channel URLs each week.
4
Create draft
Triggers a GitHub Actions workflow that builds the full HTML email server-side and creates a draft in MailerLite — fully built and ready to review and send.
Architecture
This tool (GitHub Pages) — runs in the browser. Handles fetching, thumbnail processing, and triggering the workflow. No API keys stored in the code.
GitHub Actions — two workflows run in the cloud. One checks YouTube hourly for new videos. The other is triggered by this tool and creates the MailerLite campaign server-side using GitHub Secrets.
GitHub Secrets — all sensitive credentials (YouTube API key, WordPress app password, MailerLite API key) live here. Never exposed to the browser.
